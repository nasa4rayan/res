# Cafe Restaurant La Cote Website

Multi-page modern responsive restaurant website built with HTML, CSS, and JavaScript.

## Image curation summary (from provided photos)

Selected best photos:
- Restaurant Interior: neon dining hall (best quality/composition)
- Food Dishes: plated salad dish
- Food + Interior: plated grill dish with dining background

Removed / not used:
- Dark table setup photo (noticeable blur and noise)
- Exterior night entrance shot (color cast, noise, low clarity)

The live website currently uses optimized high-quality web images (Unsplash URLs) as placeholders for production quality. You can replace them with your final curated local photos in `assets/images/`.

## Pages
- `index.html` Home
- `menu.html` Interactive menu filtering by category
- `gallery.html` Gallery with lightbox
- `contact.html` Contact map/form/CTA

## Features
- Dark premium style (black/gold/white)
- Sticky navbar + mobile menu
- Smooth scrolling + reveal animations
- Responsive cards and layouts
- Professional footer
